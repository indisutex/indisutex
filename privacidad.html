Política de Privacidad y Tratamiento de Datos

INDISUTEX S.A.S., identificada con NIT 901925743-2, domiciliada en CL 15 #7-85 Piso 2, Cali, Valle del Cauca, Colombia, actúa como responsable del tratamiento de los datos personales recolectados a través de este sitio web, en cumplimiento de la Ley 1581 de 2012 y sus decretos reglamentarios.

Datos que recopilamos

Podemos recopilar los siguientes datos cuando el usuario interactúa con nuestro sitio o canales de comunicación (WhatsApp, correo electrónico):

  • Nombre completo
  • Número de teléfono o WhatsApp
  • Correo electrónico
  • Ciudad y dirección de entrega
  • Datos de navegación mediante el Meta Pixel de Facebook (para fines estadísticos y publicitarios)

Finalidad del tratamiento

  • Atender solicitudes de información y pedidos
  • Gestionar el proceso de venta y despacho de productos
  • Enviar catálogos e información comercial de nuestras marcas
  • Mejorar la experiencia del usuario en nuestra plataforma
  • Optimizar campañas publicitarias mediante herramientas de análisis (Meta Pixel)

Transferencia de datos

INDISUTEX S.A.S. no vende ni comparte la información personal de sus usuarios con terceros con fines comerciales. Los datos únicamente se compartirán con operadores logísticos (transportadoras) para la entrega de pedidos, con las medidas de seguridad correspondientes.

Meta Pixel

Este sitio web utiliza el Meta Pixel de Facebook (ID: 703208404945102) para recopilar datos estadísticos de navegación y optimizar la publicidad en las plataformas de Meta. Al continuar navegando, el usuario acepta el uso de esta herramienta de medición. Para más información sobre las políticas de datos de Meta, visite facebook.com/privacy.

Derechos del titular

Como titular de sus datos personales, usted tiene derecho a conocer, actualizar, rectificar y suprimir su información personal. Para ejercer estos derechos, contáctenos por los canales indicados en este documento.
